Name:
  Sheepmen
Version:
  1.0
Description:
  Sheepmen sprites based on Minotaur by Jordan Irwin. Reworked by
  olonu/onu as drop-in replacements for Stendhal.
Sprite details:
  - Dimensions: 48x64
  - Orientation: orthogonal (N/E/S/W)
  - Animation: 12 frames (3 per direction)
  - PNG images use indexed color (smaller files).
